组件

组件详解

  • 组件注册
//全局注册
Vue.component('my-component',{
  template: ''
});

//局部注册
var Child = {template:''}
var app = new Vue({
  components:{
    'my-component':Child
  }
  });

  • 组件使用 <my-component></my-component>

  • 特殊组件

直接使用组件无效，必须结合　is 来挂载

<table>
    <tbody is="my-component"></tbody>
</table>
  • 组件的data 必须势函数，并return出去
Vue.component('my-component',{
  template: '<div>{{msg}}</div>',
  data: function(){
    return { msg:'组件内容'}
  }
  })

  • props(字符串数组,对象)　父组件传递数据给子组件 父组件动态传递数据用v-bind(:)

  • Vue2.x　是单向数据流，避免子组件改变父组件的状态

    <my-component :count='1' :width='100'></my-component>
Vue.component('my-component',{
props:['count','width'],
template: '<p></p>',
data:function(){
  return {
    mycount: this.count,
    mywidth: this.width + 'px'
  }
}
});
之后只用维护mycount,mywidth，可避免直接操作count,width

  • 数据验证(Type类型:String ,Number,Boolean,Array,Function,Object): instanceof检测

Vue.component('my-component',{
  props:{
    propA: Number,
    propB: String/[Sting,Number],
    propC: {
      type: Boolean,
      default: true (如果没定义，默认true)
    },
    propD:{
      type: Number,
      required:true (数字,必传)
    },
    propE:{
      type: Array,
      default:function(){
        return {} (数组或对象,默认值必须一个函数来返回)
      }
    },
    propF:{
      validator: function(value){
        return vlaue > 10; (自定义验证)
      }
    }
  }
  })
